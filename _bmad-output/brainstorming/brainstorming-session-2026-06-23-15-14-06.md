## 创意索引（共 18 项）

### 一、设计决策（ID-001 ~ ID-009）

**ID-001：多人生阶段结构**
游戏覆盖：本科 → 毕业 → 研究生/工作 → Gap 期 → 第一份工作 → 退休 → 结局。每个阶段有各自的剧情内容和机制。

**ID-002：五大核心属性（心理健康为暗主线）**
- 💰 金钱 —— 归零可借可赚，有救
- ⚡ 精力/体力 —— 归零可休息恢复，有救
- 📚 能力/技能 —— 不足可学可补，有救
- 🤝 人脉/关系 —— 断裂可修复或重建，有救
- 😊 **心理健康** —— **暗主线。唯一归零即退场的属性。** 所有重大选择都在"对自己好"和"满足外界期待"之间做 trade-off。一时迁就外界期待可获短期收益，但长期侵蚀心理——玩家要自己决定代价的边界在哪里。

**ID-003：多叙事声音**
采用"E"模式——多声部混合叙事。不同叙事声音用不同视觉样式区分。

**ID-004：拼贴/meme 视觉风格**
极简文字型 + 拼贴/Meme 美学。风格轻松、网感强、适合社交传播。

**ID-005：事件驱动叙事**
以事件为基本叙事单元，玩家通过选择驱动剧情发展。

**ID-006：多层选择系统**
日常选择 + 重大人生决策 + 随机事件 + 隐藏触发 四层并存。

**ID-007：固定背景卡 + 动态社交事件**
角色有固定背景设定，同时社交事件动态变化，两者结合。

**ID-008：混合情感基调**
戏谑幽默为主，穿插真实沉重时刻。笑完之后偶尔沉默。

**ID-009：多结局设计**
多条路线导向不同结局，结局生成"人生简历"海报。

### 二、游戏机制（ID-010 ~ ID-017）

**ID-010：社畜文化梗全收录**
- 职场类：996、OKR、复盘、优化毕业、向上管理
- 求职类：学历贬值、海投、群面、已读不回、offer被取消
- 考研考公类：二战三战、考公热、调剂
- 家庭类：催婚、催考公、别人家孩子
- 网络梗类：鼠鼠我呀、孔乙己长衫、躺平、摆烂、润
- 生理心理类：脱发、失眠、体检异常、出租屋孤独想哭

**ID-011：出租屋孤独时刻**
深夜 + 低社交 + 高压力条件触发。安静叙述画面，无操作选择。心理健康值高/低触发不同独白。贯穿全文的仪式感情感锚点。

**ID-012：命运骰子系统**
重要事件由随机 roll 决定（1-3 坏结果，4-6 好结果），玩家控制的是面对结果后的反应选择。强化时间随机性。

**ID-013：梗力系统**
meme/名场面作为正式机制。正确时机接梗得 buff，该严肃时玩梗翻车。考验玩家的情境阅读能力。

**ID-014：人生朋友圈**
每个阶段结束自动生成朋友圈动态。虚拟好友回复随选择变化。既是阶段回顾，也是笑点载体。

**ID-015：简历人生**
每个重要选择写入"人生简历"。结局生成完整简历海报（可保存/分享）。既是个人回顾，也是社交传播点。

**ID-016：坏人路线**
可选择做坏人——PUA 下属、给同事穿小鞋、靠关系上位。不同道德路径导向不同结局，不评判只展示。

**ID-017：彩蛋结局**
极低概率触发的隐藏结局，供玩家挖掘和分享。例：外星人抓走、中彩票、发现自己是 NPC。

### 四、核心设计参数

| 参数 | 值 |
|------|-----|
| 单局时长 | 15-20 分钟 |
| 心理健康暗主线 | **心理健康为唯一退场属性。** 其他属性崩了都能救（缺钱可借、体力可休、能力可学、人脉可修）。心理健康归零 → 角色以该方式退场 → 触发该路线特殊结局。所有重大选择都是"对自己好" vs "满足外界期待"的 trade-off |
| 内容边界 | 百无禁忌，幽默自洽 |
| 现实感 | 虚构但"懂的都懂"，数字尽量真实 |
| 周目差异来源 | 学历路径 × 家庭背景 × 时间随机性 |
